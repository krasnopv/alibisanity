# Alibi Sanity Studio

Sanity Content Studio for Alibi Studios project.

## Quick Start

### Production (Default)

```bash
npm run dev          # Start development server (production dataset)
npm run build        # Build for production
npx sanity deploy    # Deploy to Sanity.io
```

### Staging

```bash
npm run dev:staging  # Start development server (staging dataset)
npm run build:staging # Build for staging
```
